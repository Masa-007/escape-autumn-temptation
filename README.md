# Escape Autumn Temptation 🍂🎮

秋の食欲の誘惑から逃げる脱出ゲーム

## 🎯 ゲーム概要
プレイヤーが秋の美味しい食べ物の誘惑を振り切って、制限時間内にゴールを目指すアクションゲームです。

## 🚀 技術スタック
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📝 開発予定機能
- [ ] プレイヤーの移動機能
- [ ] 食べ物オブジェクトの配置
- [ ] 衝突判定システム
- [ ] スコア・タイマー機能
- [ ] ゲームオーバー・クリア判定

## 🎮 操作方法
- 矢印キー：プレイヤー移動
- スペースキー：ゲーム開始/一時停止
